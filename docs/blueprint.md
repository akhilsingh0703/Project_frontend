# **App Name**: UniFriend

## Core Features:

- University Search: Global search for universities by name, location, and program.
- Filtering & Sorting: Filter universities by country, program, budget, acceptance rate, and sort by tuition, popularity, etc.
- University Comparison: Compare up to 3 universities side-by-side on key attributes.
- University Details: Comprehensive profile pages with overview, programs, tuition, campus life, admissions, and map.
- Content Generation: Use an AI tool to summarize information about Universities.
- User Sharing: Enable users to share university comparison reports via email.

## Style Guidelines:

- Primary color: Light blue (#A7D1FF) for a calm, trustworthy feel, hinting at higher education.
- Background color: Off-white (#F5F9FF), a desaturated tint of the primary, for a clean and modern look.
- Accent color: Indigo (#4B0082) for key CTAs and highlights to create contrast.
- Body font: 'PT Sans', a humanist sans-serif, providing a modern and slightly warm feel.
- Headline font: 'Space Grotesk' sans-serif for headings to ensure clear legibility and a modern feel.
- Use minimalistic icons to represent university features and filter options.
- Employ a card-based design for university listings and comparison tables for a clean, organized UI.
- Subtle transitions and animations to enhance user experience when filtering and comparing.
